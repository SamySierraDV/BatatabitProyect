## ✨ Características

- 📱 **Diseño Mobile-First** — adaptable a todos los dispositivos
- 🎨 **CSS3 puro** — sin frameworks, con variables y buenas prácticas
- 🧱 **HTML5 semántico** — estructura accesible y clara
- 💹 Secciones: Hero, Tabla de tasas, Beneficios, Planes y Footer
- ⚡ Desplegado en **GitHub Pages**
